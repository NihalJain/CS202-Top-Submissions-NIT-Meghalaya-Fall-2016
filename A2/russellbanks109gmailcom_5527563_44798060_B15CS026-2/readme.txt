The program is executed by clicking the Run icon or pressing F9.
The program takes integer values which are non repeating.
Enter an integer value for the number of soldiers.
Enter an integer value which will act as the sequence in which the soldiers will be killed.
Enter a single character variable or symbol to represent the name of the soldier. 
For example,
Enter the number of soldiers:6
Enter the number sequence:3
Enter the name of the soldiers:A B C D E F
